# ChatCore Chat App Example

A small, runnable chat app built around the `chatcore` engine.

ChatCore provides the durable room/event/sync primitives. This example supplies
the host-app pieces around it:

- a TypeScript Node HTTP server for transport
- Server-Sent Events for live delivery
- a TypeScript browser UI for room creation, joining, and sending messages
- a persistent SQLite database stored at `data/chatcore.sqlite`
- demo-only in-process SSE client tracking

The example uses `better-sqlite3` and Kysely directly. The host app provisions
ChatCore's five tables from CLI-generated SQL, implements ChatCore's storage
interface with Kysely, then passes that storage object into `createChatCore`.

## Run

From the repository root:

```bash
pnpm install
pnpm -C examples/chat-app dev
```

Then open `http://localhost:5173`.

The source lives in `src/server.ts` and `src/client.ts`. `pnpm -C
examples/chat-app build` compiles them to `dist/`, and the server serves the
compiled browser script at `/app.js`.

Delete `examples/chat-app/data/chatcore.sqlite` to reset the demo database.

Set a display name in two browser windows and join the same room to try a live
conversation.

## Schema Generation

The checked-in SQLite schema at `schema/chatcore.sql` is generated by the
separate `@chatcore/cli` package:

```bash
pnpm -C examples/chat-app schema:generate
```

That script runs:

```bash
chatcore schema generate --dialect sqlite --out schema/chatcore.sql
```

The server applies `schema/chatcore.sql` automatically when
`data/chatcore.sqlite` does not contain the ChatCore tables yet. The same
integration can be run explicitly with `db:init`, which opens the real SQLite
database and executes the generated SQL:

```bash
pnpm -C examples/chat-app schema:generate
pnpm -C examples/chat-app db:reset
pnpm -C examples/chat-app db:init
pnpm -C examples/chat-app dev
```

The integration code is intentionally small:

```ts
const schemaSql = await readFile(schemaPath, "utf8");
db.exec(schemaSql);
```

See `src/schema.ts` for the full helper, including table-existence checks and
SQLite foreign-key setup.

This only demonstrates schema provisioning. The example still implements
`ChatCoreStorage` itself in `src/server.ts`, including JSON serialization,
row decoding, and storage filter translation.
